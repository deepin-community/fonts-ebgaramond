EB Garamond Italique

Les styles de Robert Granjon en open source

L’italique qui accompagne EB Garamond est basée sur le style de Robert Granjon, imprimé sur le Spécimen de Berner. Depuis la version 0.014, l’italique est comprise dans les téléchargements disponibles sur github.
